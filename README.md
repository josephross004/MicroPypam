# MicroPypam: Stripped PyPam for use on embedded 32-bit system

## ACKNOWLEDGEMENTS
The [PyPam](https://github.com/lifewatch/pypam), [PyHydrophone](https://github.com/lifewatch-pyhydrophone) projects contributed greatly to this effort.
